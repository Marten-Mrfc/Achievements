# Totaal voorbeeld
Hier een totaal voorbeeld van een achievement met alle opties erin verwerkt.
<Tabs groupId='language' queryString>
  <TabItem value="java" label="Java" default>
    ```java
    AchievementAPI.register("voorbeeld", builderConsumer -> {
        builderConsumer.title("De titel"); // verplicht
        builderConsumer.description("Een beschrijving"); //verplicht
        builderConsumer.icon(Material.BEDROCK);
        builderConsumer.showToast(true); // Standaard true
        builderConsumer.hidden(false); // Standaard false
        builderConsumer.frame(AdvancementDisplay.Frame.CHALLENGE); // Standaard TASK
        builderConsumer.parent("root"); // Optioneel
    });
    ```
  </TabItem>
  <TabItem value="kotlin" label="Kotlin">
    ```kotlin

    }
    ```
</TabItem>
