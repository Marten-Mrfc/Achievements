# Extras
Op deze pagina leer je wat handige extra features van een achievement builder.

## Parents
Je hebt nu een achievement aangemaakt maar hoe link je nou achievement A met achievement B? Dat doe je met parents.
<Tabs groupId='language' queryString>
  <TabItem value="java" label="Java" default>
    Om een parent toe te voegen in java gebruik je:
    ```java
    builderConsumer.parent("voorbeeld");
    ```
  </TabItem>
  <TabItem value="kotlin" label="Kotlin">
    Om een parent toe te voegen in kotlin gebruik je:
    ```kotlin
    parent("voorbeeld")
    ```
</TabItem>
</Tabs>
parents zijn optioneel, je kan dus ook een achievement zonder parent aanmaken.

## Frame
Een frame is het soort kader dat om het achievement heen zit. Er zijn drie soorten frames:.

  <TabItem value="java" label="Java" default>
    Om een frame toe te voegen in java gebruik je:
    ```java
    builderConsumer.frame(AdvancementDisplay.Frame.TASK);
    ```

## Hidden
Je kan een achievement ook verbergen. Dit betekent dat het niet zichtbaar is in de lijst met achievements totdat je het behaalt. 
:::info
Het liefst wil je als speler alle achievements zien daarom default deze waarde naar false!
:::
<Tabs groupId='language' queryString>
  <TabItem value="java" label="Java" default>
    Om een achievement te verbergen in java gebruik je:
    ```java
    builderConsumer.hidden(true);
    ```
  </TabItem>
  <TabItem value="kotlin" label="Kotlin">
    Om een achievement te verbergen in kotlin gebruik je:
    ```kotlin
    hidden(true)
    ```
</TabItem>
</Tabs>
